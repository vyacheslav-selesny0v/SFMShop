GET /products - список товаров:

    HTTP метод: GET
    Путь: /products
    Описание: Получние все доступных товаров
    Параметры (path parameters, query parameters):
        - id (integer): идентификатор товара
        - name (string): название товара
        - price (integer): цена товара
    Статусы ответов (успех и ошибки):
        - 200 OK: запрос выполнен успешно
        - 400 Bad Request: некоректные параметры запроса
        - 404 Not Found: товар не найден
        - 500 Internal Server Error: ошибка на стороне сервера
    Примеры запросов и ответов:
        GET /products HTTP/1.1
        Host: sfmshop.com

        Ответ:
        HTTP/1.1 200 OK
        Content-Type: application/json
        [{"id": 1, "name": "Ноутбук", "price": 50000}, ...]
        ----------
        400 Bad Request:
        { "error": "Invalid query parameter", "message": ...}
        ----------
        404 Not Found:
        { "error": "Not Found", "message": ...}
        ----------
        500 Internal Server Error:
        { "error": "Internal Server Error", "message": ...}
